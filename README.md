# Chess tournament projection
## Start

### first clone the project

### Windows
````bash
$ git clone https://github.com/Jogodev/chess_tournament.git
$ cd chess_tournament
````
### Mac
````bash
$ git clone https://github.com/Jogodev/chess_tournament.git
$ cd chess_tournament
````
### Linux
````bash
$ git clone https://github.com/Jogodev/chess_tournament.git
$ cd chess_tournament
````

### Install the virtual environment
````bash
$ pip install -r requirements.txt
````
